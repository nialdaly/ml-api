# ml-api
A pattern for deploying machine learning models as API endpoints. This pattern utilises the following tools:
- HuggingFace
- FastAPI
- Amazon SageMaker
- Terraform

To get started, run the following commands in the project root:
```
python3 -m venv .venv
source .venv/bin/activate
pip3 install -U pip
pip3 install poetry
poetry install
```
